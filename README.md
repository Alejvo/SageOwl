# SageOwl

## Technologies
- .NET 9 (Asp.NET Core)
- ASP.NET MVC
- Redis
- Docker
- SQl Server

## Purpose
SageOwl is a personal project that could be useful in academic environments, 
the application provides a team system divided by a role (admin, member), within 
which users can post announcements, admins can manage team qualifications and create and update
forms, in addition, admins can review the results of the forms.
